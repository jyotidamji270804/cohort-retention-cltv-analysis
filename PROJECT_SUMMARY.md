# Project Summary

## Project Name
Cohort Retention & Customer Lifetime Value (CLTV) Analysis

## Objective
Analyze customer retention patterns and calculate Customer Lifetime Value using transactional retail data.

## Dataset
Online Retail Dataset

## Work Completed

### Data Cleaning
- Removed missing Customer IDs
- Removed cancelled transactions
- Converted InvoiceDate to datetime
- Created Sales column

### Cohort Analysis
- Created InvoiceMonth
- Created CohortMonth
- Calculated CohortIndex
- Built Retention Matrix
- Calculated Retention Rate
- Generated Retention Heatmap

### CLTV Analysis
- Calculated Average Order Value (AOV)
- Calculated Purchase Frequency
- Calculated Customer Lifetime Value (CLTV)
- Identified Top Customers

## Key Findings

- Customer retention decreases over time.
- Most customer churn occurs after the first month.
- High-value customers contribute a large share of revenue.
- Improving retention can significantly improve CLTV.

## Business Recommendations

- Implement customer retention campaigns.
- Develop loyalty programs.
- Re-engage inactive customers.
- Focus marketing on high-value customers.
